Tugas Function Dart

Deskripsi Project
Project ini dibuat menggunakan bahasa Dart untuk memahami konsep function dengan 3 topik utama:
1. Topik Menyapa
2.  Topik Kalkulator Luas Persegi
3. Topik Kalkulator Diskon

Setiap topik memiliki 4 jenis function:

* Function Declaration
* Function Expression
* Arrow Function
* Function dengan Return Value

---

Cara Kerja Project
1. Topik Menyapa
Menampilkan pesan sapaan ke user, termasuk menggunakan parameter nama.

2. Kalkulator Luas Persegi
Menghitung luas persegi dengan rumus:
Luas = sisi × sisi

3. Kalkulator Diskon
Menghitung harga setelah diskon:
Harga akhir = harga - (harga × persen / 100)

---

Teknologi

* Dart

---

Cara Menjalankan
1. Install Dart SDK
2. Buka terminal
3. Jalankan perintah:
   dart run main.dart

---

Tujuan
* Memahami function di Dart
* Menggunakan parameter & return value
